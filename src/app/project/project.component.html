Melispot Analytics

melispot.com.br

Melispot is an analytics tool developed for Mercadolivre sellers, offering real-time information on sales performance. It helps sellers keep track of important metrics like revenue in different categories, top-selling and underperforming products, and the results of changes in product descriptions or prices.

Easy to integrate through OAuth, Melispot provides practical data that sellers can use to understand market trends and improve their sales tactics.

Self-Service Phishing Simulations Platform

wunderdigital.com.br

Wunder Cyber is a cutting-edge self-service platform designed to enhance cybersecurity awareness. It enables clients to effortlessly set up phishing simulations, leveraging customizable email templates.

The platform provides insightful dashboards to track key metrics such as email sends, opens, clicks, and instances of sensitive data entry, thereby measuring human risk vulnerability.

Wunder Digital - Microlearning

wunderdigital.com.br

Wunder microlearning was the first product launched at Wunder Digital. It is a mobile-first e-learning platform tailored for large, distributed teams. It provides real-time analytics to monitor how users engage with the content and how they progress.

My role in this project was integral, I was responsible for designing the architecture and developing the serverless backend.

Additionally, I crafted the web interface using Angular and built the Android app with Java and Kotlin. For the iOS application, I hired an external resource.
